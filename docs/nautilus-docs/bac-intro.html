Basics of Coding with Nautilus

Intro

Nautilus is a programming language that is designed to be easy to learn and use.

We use Google's block coding system, Blockly, for creating easy to understand blocks of code. You have probably seen blocks used before, maybe in Scratch or Microsoft MakeCode.

Nautilus uses Blockly to create blocks that can be used to create programs that do things like respond to messages, give members roles, or just about anything else you can do in JavaScript.


What is code?

Simply put, coding is used for communicating with computers. People use coding to give computers and other machines instructions on what actions to perform. Further, we use it to program the websites, apps, and other technologies we interact with every day.

This brings us to the next question: how does coding work?

The answer is simple:

You write code.

You compile the code.

You run the code.

It either works, or doesnt. Back to step 1.


Ok, but how does this tie into Nautilus?

As mentioned before, Nautilus uses blocks to code. These blocks are translated to JavaScript, just like how you can translate English to French.

The JavaScript code is then run on the computer. The computer then does what the JavaScript code says it should do.

This is the same process that we use to write code in our apps, websites, and other technologies. Except most of these don't use blocks.
