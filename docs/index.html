The Valour VAPP Project

The easiest way to make bots for Valour.

Valour VAPP is a open source, community built project, with the goal of making it extremely easy to make bots for Valour. Using block code (like Scratch!), it is possible to build-a-bot for Valour in less than 5 minutes. Nautilus is our program used to do all of this, from setting up your bot account, to making your first bot, all the way to executing your code and having it communicate with Valour. Valour VAPP is currently in early alpha, everything here is subject to change.

Valour VAPP is not associated with Valour, Vooper Media, or any other Valour thing whatsoever. This is a project made by the community of Valour.
