Disclaimer: This page only talks about the main menu (seen below). For info on the block editor's UI, vist here.

Home menu buttons

  • Start

    • New bot opens the Block Editor.

    • "Node bot" opens the node editor

    • There are also a few examples.

  • Bot settings

    • "Edit login info" brings you to the settings page, to edit the username and password used to log in to Valour remotely.

    • "View stored data" Lets you view all of the data stored locally.
