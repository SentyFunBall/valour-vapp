Basics of Coding - Logic

Logic

Logic is the process of deciding what to do based on the information you have.

What is logic?

Boolean logic, shortened to just "logic" here, are stored as either On or Off in memory. True or False. 0 or 1.

Computers can decide weather or not to do something based on if the logic is true or false. For example, if your age is less than 10, you get the ice cream for free. Otherwise, the ice cream costs $10.

There are several operators when dealing with logic: AND, OR and NOT. These can be used to combine logic statements to allow for more conditionality.

Example: if your age is less than 19 AND your age is greater than 10, the ice cream costs $5. The ice cream will only be $5 if your age is between 11 and 18.

There are a few more operators as well: Less than, greater than, and equal to.

Less than is true if A is less than B (10 is less than 15). If A is not less than B, the statement will be false (15 is less than 10)

Greater than is true if A is greater than B (15 is greater than 10). If A is not greater than B, the statement will be false (10 is greater than 15)

Equal to is true if A is equal to B (10 is equal to 10). If A is not equal to B, the statement will be false (10 is equal to 15)

Logic in Nautilus

In Nautilus, all of the explained logic conditions exist. There is a single block for comparing two things.

The "=" block can be configured for less than, greater than and NOT equal as well.

The "AND" block can be configured for AND and OR.

if else if else

There is more to the logic story than that though. We can configure the if block to have multiple conditions, and do different things depending on which condition is true.

We can complete our ice cream story with this. We want it so if our age is less than 10, the ice cream is free. If our age is between 10 and 18, it costs $5, and any other age it costs $10.

This is a multi-step process, so lets go through and explain how else if statements work.


The else statement

The else statement is used to run code if none of the conditions are true. This is useful for when you want to run code if none of the conditions are true.

Example: our code checks if age is less than 10. If age is 15, then the if statement would be false. With an else statement added on, the code inside the else would run if none of the conditions in the if statement are true.


The else if statement

The else if statement is used to extend conditionality of the if statement. It acts like a chain: If the first if statement returns false, the second if statement, called the else if, is checked. If that is true, the code inside it is run, otherwise it runs to the next link in the chain.

Example: our code checks if age is less than 10. If age is 15, then the if statement would be false. With an else if statement added on, we can check if the age is something other than less than 10. We can check for if age is between 10 and 18 with this.
