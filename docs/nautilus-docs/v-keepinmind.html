Welcome to the Nautilus IDE Documentation.

Nautilus, and The VAPP Project, have no affiliation with Valour, except for the ability to send and recive messages while not logged in.

While we have tested our program many, many times in many many different ways, it is not possible to test every code combination on every planet with every user. Some code may not work, while some may break your program.

We have safeguards in place to prevent Nautilus from crashing your computer, and from crashing Valour. If you find an issue with Nautilus, please submit an issue either on the official repository, or on the Discord server.
