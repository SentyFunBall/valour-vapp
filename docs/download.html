Nautilus

Nautilus is the IDE we made just for editing and execute the code for Valour bots. It is extremely easy to use and can make coding bots for Valour easy as pie. Just fire it up, click the "Block Projects" button and start making bots! You can also export your code projects to run them on a server or another computer.


How it works

Nautilus is really simple how it works. It uses Google Blockly as the foundation for the blocks, and we have built on that. It communicates with Valour with the API, which lets us chose where messages can go, based off what the user says.
