For loops

The count with block has four parameters: which variable to count with, set to i (iteration) by default, the number to start with, the number to end with, and how much to change the variable each iteration.

These can all be changed, and set to other variables to make really dynamic and fun loops. There is also a much more simple version of this that just repeats the code block x times.

While loops

The while block has one parameter, the condition to check. It will repeat the code block until the condition is met.

Lists

Lists are a way to store multiple items in one variable. Lists are created in Nautilus by using the create list with block. It can be configured to store as many items as you want.

Lists can store any type of data, and can be used in many different ways. For example, you can store a list of numbers, and then use the for each item in list loop to print out each number in the list, as shown previously. You could also store strings, other lists, or mix them all up.
