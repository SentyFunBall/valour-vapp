Valour bot Application Project

The easiest and simpliest way to make bots for Valour.

Valour VAPP is a open source, community built project, with the goal of making it extremely easy to make bots for Valour. Using block code (like Scratch!), it is possible to build-a-bot for Valour in less than 5 minutes. Nautilus is our program used to do all of this, from setting up your bot account, to making your first bot, all the way to executing your code and having it communicate with Valour. Valour VAPP, Nautilus is currently in beta, everything here is subject to change.

VAPP is an open-source project. Check out the github here.
